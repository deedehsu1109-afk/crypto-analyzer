# CryptoAnalyzer 安裝教學（Windows / macOS）

> 本文件說明如何在**另一台電腦**上從 GitHub 取得並啟動 CryptoAnalyzer。
> 若電腦上已經裝好系統，操作方式請改看系統內建的「📖 操作手冊」。

`crypto-analyzer` 是**私有 GitHub 倉庫**，兩個平台的第一步都必須先完成 GitHub 授權，否則 `git clone` 會直接失敗（顯示 repository not found）。

---

## 系統需求（兩平台通用）

| 項目 | 版本 |
|---|---|
| Python | 3.11 以上（建議 3.14，與開發環境一致） |
| Git | 需能存取此私有倉庫的帳號授權 |
| Tesseract OCR | 選用，僅辨識掃描型 PDF 時需要 |

---

## 一、Windows 安裝步驟

### 1. 安裝 Python
到 [python.org](https://www.python.org/downloads/) 下載安裝，**安裝畫面務必勾選「Add python.exe to PATH」**，否則後面指令找不到 python/pip。

### 2. 安裝 Git
到 [git-scm.com](https://git-scm.com/download/win) 下載安裝，一路預設值即可。

### 3. 設定 GitHub 存取權限（私有 repo 必做，擇一）

**方法 A｜HTTPS + Personal Access Token（較簡單）**
1. 瀏覽器登入 GitHub → 右上角頭像 → Settings → Developer settings → Personal access tokens → 建立一個有 `repo` 權限的 token，複製起來
2. 之後 `git clone` 時會跳出登入視窗，帳號填 GitHub 帳號、密碼欄貼上剛才的 token

**方法 B｜SSH Key**
1. 開啟 PowerShell，執行：
   ```powershell
   ssh-keygen -t ed25519 -C "你的信箱"
   ```
   一路按 Enter 使用預設路徑
2. 用記事本開啟 `C:\Users\<你的帳號>\.ssh\id_ed25519.pub`，複製全部內容
3. 貼到 GitHub → Settings → SSH and GPG keys → New SSH key
4. Clone 時網址改用 `git@github.com:...` 開頭的 SSH 網址

### 4. Clone 專案

開啟 PowerShell，切換到想放置專案的資料夾（例如 `D:\claude\`），執行：

```powershell
git clone https://github.com/deedehsu1109-afk/crypto-analyzer.git
cd crypto-analyzer
```
（若用 SSH：`git clone git@github.com:deedehsu1109-afk/crypto-analyzer.git`）

### 5. 安裝 Python 套件

```powershell
pip install -r requirements.txt
```

### 6.（選用）安裝 Tesseract OCR

只有需要辨識**掃描型 PDF**（無法直接抓文字的影像式筆錄）才需要，可以先跳過，系統偵測到需要時會自動提示下載連結。手動安裝：到 [UB-Mannheim/tesseract](https://github.com/UB-Mannheim/tesseract/wiki) 下載 Windows 安裝檔，安裝時**記得勾選 `chi_tra`（繁體中文）語言包**。

### 7. 啟動系統

```powershell
python main.py
```

首次啟動會自動建立全新空白資料庫，直接可用。

## 二、macOS 安裝步驟

### 1. 安裝 Xcode Command Line Tools（內含 Git）

開啟「終端機」（Terminal），執行：
```bash
xcode-select --install
```
跳出視窗後點「安裝」，跑完即完成（macOS 內建的 Git 就在這裡）。

### 2. 安裝 Homebrew（若尚未安裝）

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
安裝完照畫面指示把 `brew` 加入 PATH（通常會提示要執行的指令，複製貼上執行即可）。

### 3. 安裝 Python（含 Tkinter 支援）

```bash
brew install python@3.12 python-tk@3.12
```
> **重要**：CustomTkinter 需要 Tkinter 支援，Homebrew 的 Python 預設不一定會帶，務必額外裝 `python-tk@3.12`（版本號要跟 `python@3.12` 對應），否則啟動時會出現 `ModuleNotFoundError: No module named '_tkinter'`。

### 4. 設定 GitHub 存取權限（私有 repo 必做，擇一）

**方法 A｜HTTPS + Personal Access Token**
同 Windows 說明：GitHub 建立有 `repo` 權限的 token，clone 時密碼欄貼上 token。

**方法 B｜SSH Key**
```bash
ssh-keygen -t ed25519 -C "你的信箱"
cat ~/.ssh/id_ed25519.pub | pbcopy    # 直接複製到剪貼簿
```
把複製的內容貼到 GitHub → Settings → SSH and GPG keys → New SSH key，之後 clone 用 SSH 網址。

### 5. Clone 專案

```bash
cd ~/Documents          # 或任何你想放置專案的資料夾
git clone https://github.com/deedehsu1109-afk/crypto-analyzer.git
cd crypto-analyzer
```

### 6. 建立虛擬環境並安裝套件（建議）

```bash
python3.12 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
> 之後每次要執行系統前，記得先 `source venv/bin/activate` 啟用虛擬環境。

### 7.（選用）安裝 Tesseract OCR

```bash
brew install tesseract tesseract-lang
```
`tesseract-lang` 會一併裝好含繁體中文在內的完整語言包。

### 8. 啟動系統

```bash
python3 main.py
```

## macOS 已知限制

以下功能在目前版本僅支援 Windows，Mac 上會自動略過或提示改用其他方式，**不影響其他功能正常使用**：

| 功能 | Windows | macOS |
|---|---|---|
| 交易所調閱申請書輸出成 **PDF** | ✅ | ❌ 會跳出錯誤訊息「找不到 Windows 中文字型，請改用 DOCX 或 ODT 格式」——**改選 DOCX 或 ODT 格式輸出即可，內容完全相同** |
| 匯入 **`.doc`**（舊版 Word 97-2003 格式）筆錄檔案 | ✅（透過 Word COM 自動化） | ❌ 靜默略過（不會報錯，但也不會擷取到內容）——**請將檔案另存為 `.docx` 後再匯入** |
| 介面中文字型（標楷體／Microsoft JhengHei／Consolas） | 使用系統內建字型 | 這些是 Windows 專屬字型，Mac 上會自動改用系統預設字型顯示，**外觀略有差異但功能不受影響** |

其餘功能（地址查詢、幣流圖、Excel/CSV 匯出、DOCX/ODT 報告產製、網站溯源、OCR 等）Windows 與 macOS 完全相同。

## 三、不會跟著 git clone 一起過去的檔案

以下兩個檔案因含敏感資料，被 `.gitignore` 排除，**新電腦上都是空的**：

| 檔案 | 內容 | 新電腦怎麼處理 |
|---|---|---|
| `config.json` | API Key、使用者身分紀錄 | 開啟系統後在右上角「⚙ 系統設定」重新填入 |
| `crypto_data.db` | 所有案件、錢包、交易紀錄 | 全新空白資料庫；要帶既有案件資料過去，需用隨身碟/雲端硬碟等方式**手動複製**這個檔案到同樣的 `crypto_analyzer/` 資料夾（不能透過 git） |

## 四、之後如何更新到最新版本

系統內建更新機制：開啟後在「使用者身分」頁面按「🔄 檢查更新」，會自動比對並詢問是否套用；也可以直接在專案資料夾手動執行：

```bash
git pull origin main
```
